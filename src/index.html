  1. 小羊记账 2.0

    小羊记账2.0是一款基于Vite3、Vue3、TSX、Pinia的一款移动端记账项目。

    在项目开发中，使用了 CSS Modules 模块化的样式。封装了Icon组件、Button组件、表单组件等十多个组件。使用插槽对组件进行封装，提高了代码的复用率和可读性；在制作欢迎页面时，使用到了动画和手势。并且使用了第三方的 Vant UI 作为 UI 组件库，引入了 Dialog、 Toast 等丰富的组件样式。通过 v-model 对表单进行双向绑定。使用了 Vue Router 进行路由配置，实现了页面之间的跳转和传参。为了保证项目的安全性和数据的安全性，使用了鉴权守卫，调用后端的接口，根据后端的返回，来提示用户登录等等。和使用 JWT 来实现身份认证。并且使用了 Axios 对 HTTP 请求进行封装，统一管理请求和响应的格式和状态码。同时，实现了一个简单的 Mock，可以篡改响应以及数据的生成，以及使用 Faker 来模拟数据进行开发。同时增加了表单验证。最后，通过可视化 一目了然的知道账单明细，提升了用户的交互体验。在增强数据安全的同时也增强了用户的体验。
